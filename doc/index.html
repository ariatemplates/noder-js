What

noderJS is a javascript loader for browsers based on CommonJS API that let you load all dependencies you need inside your web application in the same way you do in a nodejs application.

Why

As nodejs is increasingly being used to implement web servers and command line tools, it is important to be able to reuse some libraries in a browser. With noderJS you'll be able to run unmodified nodejs modules.

How

noderJS is compatible with the synchronous require syntax and, by default, performs all the requests asynchronously through XHR.
